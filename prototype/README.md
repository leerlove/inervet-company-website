# INERVET 홈페이지 HTML 프로토타입

## 개요
INERVET 웹사이트의 디자인 컨셉을 검증하기 위한 HTML/CSS 프로토타입입니다.
실제 개발 명세서, 와이어프레임, 디자인 가이드를 기반으로 제작되었습니다.

## 파일 구조
```
prototype/
├── index.html          # 홈페이지 프로토타입 (메인 파일)
└── README.md           # 본 문서
```

## 실행 방법

### 방법 1: 직접 열기
1. `index.html` 파일을 더블클릭하여 브라우저로 엽니다
2. 또는 브라우저에서 파일 > 파일 열기 > `index.html` 선택

### 방법 2: Live Server 사용 (권장)
1. VS Code에서 `index.html` 파일을 엽니다
2. 오른쪽 클릭 > "Open with Live Server" 선택
3. 자동으로 브라우저가 열리며, 파일 수정 시 자동 새로고침됩니다

### 방법 3: 로컬 서버
```bash
# Python 3 사용
cd prototype
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

## 구현된 기능

### 페이지 섹션
- ✅ **Header**: 고정 헤더, 네비게이션, CTA 버튼
- ✅ **Hero Section**: 메인 헤드라인, CTA, 신뢰 지표
- ✅ **Key Metrics**: 주요 지표 4개 (18,000+ 상담 등)
- ✅ **Problem Section**: 사용자 페인 포인트 3개
- ✅ **How It Works**: 4단계 프로세스 설명
- ✅ **Pricing**: 기본/프리미엄 요금제
- ✅ **Testimonials**: 고객 후기 3개
- ✅ **Final CTA**: 마지막 전환 유도
- ✅ **Footer**: 링크, 소셜 미디어, 회사 정보

### 디자인 시스템
- ✅ **컬러**: Primary Blue (#1E3A8A), Accent Purple (#8B5CF6)
- ✅ **타이포그래피**: Pretendard (한글), Inter (영문 대체)
- ✅ **스페이싱**: Tailwind 4px 기반 시스템
- ✅ **컴포넌트**: 버튼, 카드, 배지, 아이콘 등

### 인터랙션
- ✅ **애니메이션**: Fade-in, slide-up 효과
- ✅ **Hover Effects**: 버튼, 카드 hover 시 변화
- ✅ **Smooth Scroll**: 앵커 링크 부드러운 스크롤
- ✅ **Scroll to Top**: 상단으로 이동 버튼

### 반응형 디자인
- ✅ **Desktop**: 1024px 이상
- ✅ **Tablet**: 768px - 1023px
- ✅ **Mobile**: 767px 이하
- ✅ **Breakpoints**: Tailwind 기본 브레이크포인트 사용

## 기술 스택
- **HTML5**: 시맨틱 마크업
- **Tailwind CSS 3.x**: CDN 방식 (프로토타입용)
- **Pretendard Font**: CDN 방식
- **Vanilla JavaScript**: 스크롤 인터랙션

## 프로토타입 검증 포인트

### 디자인 검증
- [ ] 브랜드 컬러가 일관되게 적용되었는가?
- [ ] 타이포그래피 크기와 위계가 적절한가?
- [ ] 스페이싱이 시각적으로 균형잡혀 있는가?
- [ ] 아이콘과 일러스트 스타일이 통일되었는가?

### UX 검증
- [ ] Hero 섹션이 3초 내 서비스를 이해할 수 있는가?
- [ ] CTA가 눈에 띄고 클릭하고 싶은가?
- [ ] 콘텐츠 흐름이 자연스러운가?
- [ ] 주요 정보 (18,000건, 98% 등)가 효과적으로 전달되는가?

### 반응형 검증
- [ ] 모바일에서 콘텐츠가 읽기 쉬운가?
- [ ] 터치 타겟 크기가 충분한가? (최소 44x44px)
- [ ] 이미지와 레이아웃이 적절히 변환되는가?

### 접근성 검증
- [ ] 색상 대비가 충분한가? (WCAG 2.1 AA)
- [ ] 키보드로 모든 요소에 접근 가능한가?
- [ ] 스크린 리더 사용 시 콘텐츠가 논리적 순서인가?
