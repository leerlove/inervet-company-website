## 1. 와이어프레임 가이드

### 1.1 표기 규칙
- `[=====]`: 헤더/푸터
- `[-----]`: 섹션 구분선
- `[ Box ]`: 컨텐츠 박스/카드
- `[Button]`: CTA 버튼
- `<Image>`: 이미지/아이콘
- `[Text]`: 텍스트 블록
- `[=====]`: 네비게이션
- `|`: 세로 구분선 (열)

### 1.2 디바이스별 뷰
- **Desktop**: 1280px 이상
- **Tablet**: 768px - 1023px
- **Mobile**: 375px - 767px

## 11. 인터랙션 가이드

### 11.1 애니메이션

| 요소 | 애니메이션 | Timing | Trigger |
|------|------------|--------|---------|
| Hero Text | Fade-in + Slide-up | 0.6s | Page Load |
| CTA Button | Scale-up | 0.3s | Page Load (delay 0.8s) |
| Service Cards | Stagger Fade-in | 0.1s 간격 | Scroll (Intersection) |
| Key Metrics | Number Count-up | 1.5s | Scroll (Intersection) |
| Charts | Draw Animation | 1.0s | Scroll (Intersection) |
| Modal | Fade-in + Scale | 0.3s | Click |

### 11.2 Hover 효과

| 요소 | Hover 효과 |
|------|-----------|
| Button | Scale(1.05), Shadow 증가 |
| Card | Translate-y(-5px), Shadow 증가 |
| Link | Color 변경, Underline |
| Nav Menu | Background 변경 |
| Pricing Card | Border 강조, Shadow |

### 11.3 스크롤 효과

- **Sticky Header**: 스크롤 시 배경 흰색 + 그림자
- **Progress Bar**: IR 페이지 상단에 읽기 진행률 표시
- **Parallax**: Hero Section 배경 이미지 (선택)
- **Reveal**: 섹션 진입 시 Fade-in
