## 1. 브랜드 보이스 & 톤

### 1.1 브랜드 퍼스낼리티

INERVET의 브랜드 보이스는 다음 키워드로 정의됩니다:

- **전문적 (Professional)**: 의료 AI 기술에 대한 전문성과 신뢰성
- **혁신적 (Innovative)**: 펫헬스케어 산업의 게임체인저
- **공감적 (Empathetic)**: 반려동물과 보호자의 걱정을 이해하는 따뜻함
- **명확한 (Clear)**: 복잡한 기술을 쉽고 명확하게 설명
- **신뢰할 수 있는 (Trustworthy)**: 데이터와 실적으로 증명된 신뢰

### 1.2 톤의 적용

#### 일반 사용자 (반려동물 보호자) 대상
- **따뜻하고 공감적**: "우리도 알아요, 밤늦게 아이가 아프면 얼마나 걱정되는지"
- **친근하고 접근 가능**: "24시간, 어디서든 전문 수의사와 상담하세요"
- **안심을 주는**: "이미 18,000건의 상담으로 증명된 신뢰"
- **행동 지향적**: 명확한 CTA와 해결책 제시

#### 투자자 대상
- **전문적이고 데이터 중심**: "2.16조 원 규모의 시장, 3년 내 5배 투자 수익"
- **비전 지향적**: "펫헬스케어 산업의 디지털 전환을 선도"
- **객관적이고 실적 기반**: "98% 만족도, 25% 재사용률"
- **성장 가능성 강조**: "연평균 15% 성장하는 시장"

### 1.3 톤의 균형

**사용할 것:**
- ✅ 능동태 ("AI가 분석합니다" > "AI에 의해 분석됩니다")
- ✅ 구체적인 수치 ("18,000건의 상담" > "많은 상담")
- ✅ 긍정적 표현 ("빠르고 정확한" > "늦지 않고 틀리지 않는")
- ✅ 사람 중심 언어 ("보호자님의 걱정을 덜어드립니다")

**피할 것:**
- ❌ 과도한 전문 용어 (필요시 쉬운 설명 병기)
- ❌ 과장된 표현 ("세계 최고", "완벽한" 등)
- ❌ 부정적 감정 자극 ("위험합니다", "큰일납니다")
- ❌ 수동적 표현 ("~될 수 있습니다" 보다 "~합니다")

## 2. 타겟 오디언스별 라이팅 전략

### 2.1 일반 사용자 (반려동물 보호자)

#### 핵심 니즈
- 24시간 언제든 수의사 상담
- 병원 방문 전 증상 파악
- 응급 상황 판단
- 비용 절감

#### 작성 전략
1. **감정적 연결**
   ```
   예시: "밤늦게 강아지가 토하면, 지금 당장 병원에 가야 할까요?
   INERVET AI 펫닥터가 24시간 함께합니다."
   ```

2. **구체적 시나리오**
   ```
   예시: "출장 중 고양이가 밥을 안 먹는다는 연락을 받았을 때,
   언제 어디서든 전문 수의사와 영상 상담하세요."
   ```

3. **신뢰 구축**
   ```
   예시: "98%의 보호자가 만족한 상담 서비스,
   실제 수의사 18명이 24시간 대기 중입니다."
   ```

4. **명확한 혜택**
   ```
   예시: "병원 방문 전 증상을 정확히 파악하고,
   불필요한 병원 방문 비용을 줄이세요."
   ```

### 2.2 투자자

#### 핵심 니즈
- 시장 기회 (TAM/SAM/SOM)
- 비즈니스 모델 명확성
- 성장 잠재력
- 경쟁 우위
- ROI 전망

#### 작성 전략
1. **데이터 중심**
   ```
   예시: "2.16조 원 규모의 국내 펫케어 시장,
   연평균 15% 성장세로 2027년 3조 원 돌파 전망."
   ```

2. **실적 기반**
   ```
   예시: "18,000건 상담 완료, 98% 고객 만족도, 25% 재사용률로
   검증된 비즈니스 모델."
   ```

3. **차별화 포인트**
   ```
   예시: "자체 개발 AI 엔진과 실시간 수의사 네트워크 결합,
   국내 최초 24시간 AI 기반 펫헬스케어 플랫폼."
   ```

4. **성장 전략**
   ```
   예시: "Phase 1: B2C 플랫폼 확장 → Phase 2: B2B 동물병원 SaaS →
   Phase 3: 글로벌 진출 (일본, 동남아)."
   ```

## 3. 핵심 메시징 프레임워크

### 3.1 브랜드 핵심 메시지

**메인 태그라인:**
```
반려동물 건강관리의 새로운 기준, INERVET
```

**서브 태그라인 옵션:**
- "24시간, 어디서든 수의사와 함께"
- "AI가 묻고, 수의사가 답하는 펫헬스케어"
- "시간과 공간의 경계를 넘어선 동물 의료"

### 3.2 가치 제안 (Value Proposition)

#### 사용자 측면
1. **편리성**: 24시간 언제 어디서든 상담
2. **전문성**: AI 분석 + 실제 수의사 상담
3. **경제성**: 불필요한 병원 방문 감소
4. **신속성**: 평균 5분 내 상담 연결

#### 투자자 측면
1. **시장 기회**: 2.16조 원 TAM, 연 15% 성장
2. **검증된 모델**: 18,000건 상담, 98% 만족도
3. **기술 차별화**: 자체 AI 엔진 + 수의사 네트워크
4. **확장성**: B2C → B2B → Global 전략

### 3.3 핵심 차별화 포인트 (Key Differentiators)

| 구분 | 경쟁사 | INERVET |
|------|--------|---------|
| 상담 시간 | 제한적 (9am-6pm) | 24시간 연중무휴 |
| 상담 방식 | 텍스트 or 음성만 | AI 사전 분석 + 영상/음성 선택 |
| 수의사 수 | 소수 (3-5명) | 18명 이상 네트워크 |
| AI 활용 | 단순 챗봇 | 증상 분석 AI + 수의사 매칭 |
| 실적 | 미공개 | 18,000건, 98% 만족도 공개 |

## 4. 콘텐츠 구조 가이드라인

### 4.1 헤드라인 작성법

#### Hero 섹션 H1
- **길이**: 15-25자 (한글 기준)
- **구조**: [문제 인식] + [해결 제시]
- **예시**:
  ```
  ✅ "반려동물이 아플 때, 24시간 수의사와 상담하세요"
  ✅ "밤늦은 응급 상황, AI 펫닥터가 함께합니다"
  ❌ "INERVET 서비스를 이용해보세요" (너무 추상적)
  ❌ "혁신적인 AI 기반 동물 의료 플랫폼" (전문용어 과다)
  ```

#### 섹션 H2
- **길이**: 10-20자
- **구조**: [가치 제안] 또는 [사용자 질문]
- **예시**:
  ```
  ✅ "왜 18,000명이 선택했을까요?"
  ✅ "INERVET, 무엇이 다를까요?"
  ✅ "24시간 상담, 이렇게 진행됩니다"
  ```

#### 서브헤드 H3
- **길이**: 5-15자
- **구조**: [명사형] 또는 [행동 유도형]
- **예시**:
  ```
  ✅ "AI 증상 분석"
  ✅ "실시간 수의사 상담"
  ✅ "상담 후 리포트 제공"
  ```

### 4.2 본문 작성법

#### 문단 구조
- **첫 문장**: 핵심 메시지 (결론 우선)
- **중간**: 부연 설명 및 근거
- **마지막**: 다음 행동 유도 또는 다음 문단 연결

**예시:**
```
[핵심] INERVET은 24시간 언제든 전문 수의사와 상담할 수 있습니다.
[근거] 18명의 수의사가 교대로 대기하며, 평균 5분 내 상담이 연결됩니다.
[연결] 먼저 AI가 증상을 분석한 후, 가장 적합한 수의사를 매칭해드립니다.
```

#### 문장 길이
- **일반 사용자 대상**: 평균 20-30자, 최대 40자
- **투자자 대상**: 평균 30-40자, 최대 60자
- **리듬 만들기**: 짧은 문장 + 긴 문장 교차

#### 리스트 활용
**언제 사용?**
- 3개 이상의 항목 나열 시
- 단계별 프로세스 설명 시
- 혜택/기능 강조 시

**형식:**
```
✅ 좋은 예:
• 24시간 연중무휴 상담 가능
• 평균 5분 내 수의사 연결
• AI 기반 증상 사전 분석
• 상담 후 리포트 제공

❌ 나쁜 예:
• 상담이 가능합니다
• 빠릅니다
• AI를 사용합니다
• 리포트를 줍니다
(구체성 부족, 혜택 불명확)
```

### 4.3 CTA (Call-to-Action) 작성법

#### Primary CTA
- **목적**: 핵심 전환 행동 유도
- **위치**: Hero 섹션, 주요 섹션 하단
- **텍스트 길이**: 5-10자
- **구조**: [행동 동사] + [혜택/대상]

**예시:**
```
일반 사용자용:
✅ "지금 무료 상담 시작" (행동 + 혜택)
✅ "24시간 수의사와 연결" (시간 + 대상)
✅ "우리 아이 증상 확인하기" (감정 + 행동)

투자자용:
✅ "IR 자료 다운로드"
✅ "투자 문의하기"
✅ "파트너십 제안서 받기"
```

#### Secondary CTA
- **목적**: 대안 행동 제시
- **색상**: Primary보다 약한 강조
- **텍스트 예시**:
  ```
  "서비스 둘러보기"
  "가격 안내 보기"
  "FAQ 확인하기"
  ```

#### Text Link CTA
- **목적**: 추가 정보 제공
- **텍스트 예시**:
  ```
  "자세히 알아보기 →"
  "사례 더 보기 →"
  "이용 방법 확인 →"
  ```

### 4.4 마이크로카피 가이드

#### 폼 레이블
```
✅ "이메일 주소" (명확)
❌ "이메일" (간결하나 불명확할 수 있음)

✅ "회사명 (선택)" (선택 사항 명시)
❌ "회사" (필수/선택 불명확)
```

#### 에러 메시지
```
✅ "올바른 이메일 주소를 입력해주세요. (예: hello@inervet.com)"
❌ "이메일 형식이 잘못되었습니다." (해결책 없음)

✅ "연락처는 10-11자리 숫자로 입력해주세요."
❌ "입력 오류" (구체성 없음)
```

#### 성공 메시지
```
✅ "문의가 접수되었습니다. 영업일 기준 24시간 내 답변드리겠습니다."
❌ "전송 완료" (다음 스텝 불명확)
```

#### 버튼 로딩 상태
```
✅ "상담 연결 중..." (진행 중임을 명확히)
✅ "제출하는 중..."
❌ "잠시만요..." (애매함)
```

## 5. SEO 카피라이팅 가이드

### 5.1 핵심 키워드 전략

#### Primary Keywords (한국어)
- "AI 펫닥터"
- "반려동물 원격 진료"
- "24시간 수의사 상담"
- "펫헬스케어"
- "반려동물 건강 상담"

#### Secondary Keywords
- "강아지 증상 확인"
- "고양이 응급 상황"
- "수의사 온라인 상담"
- "반려동물 병원 비용"
- "펫 건강관리 앱"

#### Long-tail Keywords
- "밤늦게 강아지 토할 때"
- "고양이 밥 안먹을 때 병원"
- "반려동물 응급 상황 판단"
- "수의사 비대면 진료 가능한가"

### 5.2 Title Tag 작성법

**형식**: [핵심 키워드] | [브랜드명] - [부연 설명]

**예시:**
```html
<!-- 홈페이지 -->
<title>AI 펫닥터 INERVET | 24시간 반려동물 건강 상담 서비스</title>

<!-- 서비스 페이지 -->
<title>24시간 수의사 상담 서비스 | INERVET AI 펫닥터</title>

<!-- IR 페이지 -->
<title>투자 정보 (IR) | INERVET - 펫헬스케어 플랫폼</title>
```

**규칙:**
- 길이: 30-60자 (한글 기준)
- 브랜드명 포함
- 핵심 키워드 앞쪽 배치
- 자연스러운 문장 (키워드 나열 X)

### 5.3 Meta Description 작성법

**형식**: [문제 제시] + [해결책] + [실적/차별화] + [CTA]

**예시:**
```html
<!-- 홈페이지 -->
<meta name="description" content="반려동물이 아플 때 24시간 언제든 수의사와 상담하세요. AI 증상 분석과 실시간 영상 상담으로 18,000건 이상의 상담 완료. 지금 무료로 시작하세요.">

<!-- 서비스 페이지 -->
<meta name="description" content="INERVET의 AI 기반 펫헬스케어 서비스. 증상 입력 시 AI가 분석하고 전문 수의사가 24시간 상담합니다. 평균 5분 내 연결, 98% 만족도.">
```

**규칙:**
- 길이: 70-120자 (한글 기준)
- 핵심 키워드 자연스럽게 포함
- 구체적 수치 포함 (실적)
- 행동 유도 (CTA) 포함

### 5.4 Heading 구조 (H1-H6)

**원칙:**
- H1: 페이지당 1개 (핵심 키워드 포함)
- H2: 주요 섹션 (2-3 키워드 분산)
- H3-H4: 세부 내용 (자연스러운 문장)

**예시:**
```html
<h1>24시간 반려동물 건강 상담, INERVET AI 펫닥터</h1>

<h2>왜 18,000명의 보호자가 선택했을까요?</h2>
<h3>AI 기반 증상 분석</h3>
<h3>실시간 수의사 연결</h3>

<h2>펫헬스케어, 이렇게 달라집니다</h2>
<h3>병원 방문 전 정확한 증상 파악</h3>
<h3>24시간 언제든 전문가 조언</h3>
```

### 7.3 구두점 사용

#### 마침표 (.)
- 문장 끝에 사용
- 제목/헤드라인에는 사용하지 않음
- 리스트 항목이 완전한 문장일 때 사용

```
✅ "24시간 수의사와 상담할 수 있습니다." (본문)
✅ "24시간 수의사 상담" (헤드라인 - 마침표 없음)

✅ 리스트:
• AI가 증상을 분석합니다. (완전한 문장)
• 수의사를 자동 매칭합니다.
```

#### 쉼표 (,)
```
✅ "빠르고, 정확하고, 편리한 서비스"
✅ "증상 입력, AI 분석, 수의사 상담, 리포트 제공"
```

#### 물음표 (?)
- 헤드라인에 사용 시 사용자 공감 유도
- 본문에서 실제 질문 시 사용

```
✅ "왜 18,000명이 선택했을까요?" (헤드라인)
✅ "우리 아이가 아플 때 어떻게 하시나요?" (공감 유도)
```

#### 느낌표 (!)
**사용 최소화** - 과도하면 신뢰도 하락

```
✅ "지금 바로 시작하세요!" (강력한 CTA에만 제한적 사용)
❌ "정말 좋아요! 꼭 써보세요! 놀라울 거예요!" (과도함)
```

#### 톤 일관성
- **같은 페이지 내**: 동일한 톤 유지 (반말/존댓말 혼용 금지)
- **다른 페이지 간**: 타겟에 따라 톤 조정 가능 (일반 사용자 vs 투자자)

```
✅ 일반 사용자 페이지 전체: "~하세요", "~할 수 있습니다"
✅ 투자자 페이지 전체: "~합니다", "~입니다"

❌ 같은 페이지에서: "시작하세요" → "이용할 수 있어요" (톤 혼재)
```

### 8.4 좋은 카피 vs 나쁜 카피

#### 홈페이지 Hero

**나쁜 예:**
```
❌ H1: "INERVET에 오신 것을 환영합니다"
   → 문제: 가치 제안 없음, 서비스 불명확

❌ H1: "혁신적인 AI 기반 동물 의료 플랫폼"
   → 문제: 전문용어 과다, 사용자 혜택 불명확

❌ H1: "반려동물을 사랑하는 당신을 위한 서비스"
   → 문제: 추상적, 구체적 혜택 없음
```

**좋은 예:**
```
✅ H1: "반려동물 응급 상황, 24시간 수의사가 함께합니다"
   → 이유: 문제 제시 + 해결책, 구체적 혜택, 명확한 가치

✅ H1: "밤늦게 아이가 아플 때, 지금 바로 수의사와 상담하세요"
   → 이유: 시나리오 제시, 공감 유도, 즉시 행동 가능
```

#### 서비스 설명

**나쁜 예:**
```
❌ "저희는 최고의 기술을 보유하고 있습니다"
   → 문제: 주장만 있고 근거 없음

❌ "AI를 활용한 혁신적인 솔루션"
   → 문제: 기술 중심, 사용자 혜택 불명확

❌ "빠르고 편리한 서비스"
   → 문제: 추상적, 구체성 부족
```

**좋은 예:**
```
✅ "평균 5분 내 수의사와 연결됩니다"
   → 이유: 구체적 수치, 명확한 혜택

✅ "AI가 증상을 분석하고, 18명의 전문 수의사가 24시간 대기합니다"
   → 이유: 구체적 프로세스, 숫자로 증명

✅ "18,000건의 상담으로 검증된 신뢰"
   → 이유: 실적 기반, 사회적 증거
```

#### CTA

**나쁜 예:**
```
❌ "클릭하세요"
   → 문제: 혜택 불명확, 행동 결과 모호

❌ "더 보기"
   → 문제: 무엇을 보는지 불명확

❌ "시작"
   → 문제: 무엇을 시작하는지 불명확
```

**좋은 예:**
```
✅ "지금 무료 상담 시작"
   → 이유: 행동 + 혜택 명확

✅ "24시간 수의사와 연결"
   → 이유: 가치 제안 명확

✅ "우리 아이 증상 확인하기"
   → 이유: 감정 + 행동 명확
```

## 9. 체크리스트

### 9.1 콘텐츠 작성 전 체크리스트

- [ ] 타겟 오디언스는 누구인가? (일반 사용자 / 투자자)
- [ ] 이 페이지의 목표는 무엇인가? (인지 / 고려 / 전환)
- [ ] 핵심 메시지 3가지는 무엇인가?
- [ ] 사용자에게 취하게 할 행동은 무엇인가? (Primary CTA)
- [ ] 어떤 SEO 키워드를 타겟하는가?

### 9.2 콘텐츠 작성 후 체크리스트

#### 명확성
- [ ] 제목만 읽어도 페이지 내용을 이해할 수 있는가?
- [ ] 전문 용어를 사용했다면 쉬운 설명이 함께 있는가?
- [ ] 각 문단의 첫 문장이 핵심 메시지인가?

#### 설득력
- [ ] 주장에 대한 근거(데이터, 실적)가 있는가?
- [ ] 사회적 증거(고객 수, 만족도, 후기)가 포함되었는가?
- [ ] 사용자 혜택이 명확히 드러나는가?

#### 행동 유도
- [ ] Primary CTA가 명확한가?
- [ ] CTA가 페이지 목표와 일치하는가?
- [ ] CTA 버튼 텍스트가 행동과 혜택을 모두 담고 있는가?

#### SEO
- [ ] Title Tag에 핵심 키워드가 포함되었는가?
- [ ] Meta Description이 120자 이내인가?
- [ ] H1이 페이지당 1개인가?
- [ ] H2-H3 구조가 논리적인가?
- [ ] 이미지 Alt Text가 설명적인가?

#### 브랜드 일관성
- [ ] 톤이 타겟 오디언스에 맞는가?
- [ ] 용어가 가이드라인에 따라 통일되었는가?
- [ ] 숫자 표기가 일관적인가?
- [ ] 브랜드 보이스가 유지되는가?

## 10. 콘텐츠 리뷰 프로세스

### 10.1 자가 리뷰 (Self Review)

1. **큰소리로 읽기**: 어색한 부분이 있는가?
2. **핵심 메시지 파악**: 3초 내 무슨 내용인지 이해되는가?
3. **CTA 확인**: 다음 행동이 명확한가?

### 10.2 동료 리뷰 (Peer Review)

1. **첫인상 테스트**: 페이지를 처음 본 사람에게 "무슨 내용?"이라고 물어보기
2. **타겟 확인**: 이 콘텐츠가 누구를 위한 것인지 명확한가?
3. **행동 유도**: 읽은 후 무엇을 해야 할지 명확한가?

### 10.3 최종 승인 전 체크

- [ ] 법적 이슈 없음 (의료법, 약사법 등)
- [ ] 과장 광고 없음
- [ ] 모든 수치 출처 확인
- [ ] 오탈자 검토 완료
