## 프로젝트 개요

### 회사 소개
- **회사명**: INERVET Co., Ltd (주식회사 이너벳)
- **비전**: 반려동물 건강관리에 시간과 공간의 경계를 허무는 혁신을 추구
- **미션**: Innervation(지속 연결성)의 Innovation(혁신적인 데이터 기술)을 통해 Vet(전문 수의학)의 결합으로 미래 헬스케어 모델을 제시
- **핵심 서비스**: AI 펫닥터 - 디지털 AI 펫 헬스케어 서비스

### 홈페이지 목적
1. **일반 사용자**: 서비스 소개, 사용자 유입, 앱 다운로드 전환
2. **투자자**: 투자 유치, IR 정보 제공, 신뢰성 구축
3. **파트너**: B2B 협력 기회 제공 (병원, 아파트, 기업)

### 2. 서비스 소개 (Service)

#### AI 펫닥터란?

**개요**
AI 펫닥터는 AI 기술과 전문 수의학을 결합한 실시간 디지털 펫 헬스케어 서비스입니다.
24시간 언제 어디서나 반려동물의 건강을 관리하고, 맞춤형 상담과 영양 관리를 제공합니다.
이를 통해 언제 어디서나 반려동물의 실시간 건강관리와 조치가 가능한 시대를 만들어가고 있습니다. 

**서비스 특징**
1. **개인화 기반 AI 수의사 모델**
   - 1차 진단 및 펫 개인화 모델 집중
   - Multi Modal Agent 기반 AI 수의사 모델
   - 텍스트와 이미지를 연계하여 상담
   - RAG를 통한 검증된 DB 참조하여 답변

2. **1차 진단 최적화된 수의사 상담**
   - 초개인화 학습
   - 펫 프로필 학습 (품종, 나이, 몸무게 등)
   - 상담 히스토리 학습 (과거 상담 내역)

3. **고객 맞춤 병원 추천**
   - 고객의 상황을 고려한 병원 추천
   - 검증된 국내 4000개 이상의 병원DB
   - 위치, 상황, 전문성 기반 맞춤 추천

4. **맞춤 제품 추천**
   - 의료, 건강, 영양 등 개인화 기반의 맞춤 추천
   - 자체 검증한 안전한 제품 추천
   - 지속적인 개인화를 통한 더 명확한 추천 제공

#### 주요 기능 상세

##### 1) 24시간 AI 수의사 상담
**기능 설명**
- 내 아이를 이해하는 전담 AI 수의사와 상담
- 현재 아이의 문제를 스크리닝 방식의 빠른 1차 진단 제공
- 병원진료 필요 시 맞춤 전문 병원을 추천하고 빠른 진료 제공

**사용 플로우**
```
내 아이를 이해하는 → 전담 AI수의사와 상담 → 맞춤 병원을 추천하고 → 빠른 진료 환경 제공
```

**핵심 가치**
- 1차 진단 및 개인화 요구에 집중
- 반려동물의 70%가 원하는 1차 진단과 맞춤 상담 건강 관리에 집중
- 심리도 1위인 수의사의 맞춤 상담과 지속적인 개인화 학습으로 전담 AI 수의사 상담

**데이터 근거**
- AI 수의사 상담: 78%
- 영양/식이 상담: 33%
- 의료 질병 상담: 40%
- 행동/훈련 상담: 25%

##### 2) 맞춤 영양 관리
**기능 설명**
- 영양상태 확인하고 제품 활영을 통한 스마트 검색과 자세한 영양 정보를 확인
- 개인화 기반 영양 분석과 맞좀 영양 추천

**사용 플로우**
```
영양상태 확인하고 → 활영을 통한 검색과 → 원하는 정보를 확인 → 개인화 기반 영양 분석과 → 맞춤 영양 추천
```

**핵심 데이터**
- 제품 인식 결과: 자세한 식품 정보 제공 + 식품 위생법 기초 정보 제공 + 우수 펫푸드 제품 인증 제공
- 신규 제품 분석: 영양 상태와 연계한 주요 요청 + 제품의 배급량 + 제품 사용 기간
- 보유 제품 분석: 자세한 식품 정보 제공 + 히스토리 데이터 업데이트 + 푸시 기반 처리 과정의 밀착 케어 관리
- 품질관리팀 대응: 직접 제품 정보 확보 후 고객 제공 + 제품 정보 리서치 및 포기 전환 + 요청 고객 정보 제공

##### 3) 실시간 건강 모니터링 (AIoT)
**기능 설명**
AIoT를 통한 내 아이의 건강 변화 실시간 관리

**웨어러블 디바이스 기능**
- 기존 웨어러블 디바이스: 일상 생활 관점의 정보 제공 (호흡, 심박수, 활동량, GPS 등)
- 인바디 기술 적용 측가: 영양과 비만까지 실시간 확인 (체지방, 수분, 영양, 근육 등)

**통합 서비스**
- 인바디 펫 웨어러블 디바이스 + 건강 변화를 한눈에 확인 + 다양한 펫 디바이스와 연계

**3가지 주요 서비스**
